## Requirements
- Python 3.6+
- Chrome 84.0+
- Windows 10/macOS/Linux

## Usage
```
usage: chpass [-h] {export,import} ...

Import and Export chrome passwords

positional arguments:
  {import,export}

optional arguments:
  -h, --help            show this help message and exit
  -u USER, --user USER
```
